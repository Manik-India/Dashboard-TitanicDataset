# **Title: Dashboard on Titanic Dataset**

## **1. Description:**
 Dashboard of EDA (Exploratory Data Analysis) of Titanic Dataset using Streamlit.


## **2. How to get Output**
Just write the following command in the terminal:

```sh
streamlit run main.py
```
